## Lab 06 : Multiple Users Generator

https://o365cmu-my.sharepoint.com/:b:/g/personal/chayanin_s_cmu_ac_th1/EV_bURdOUvVLjnEP5KXwIOoB9AA8NJMaiNI18fym0UJcRA?e=YXIQeL

npm run dev
